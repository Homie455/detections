# PV Defect Detection Web App

Вебзастосунок для виявлення дефектів фотоелектричних модулів на тепловізійних зображеннях за допомогою моделі глибокого навчання YOLOv11.

## Опис проєкту

Цей проєкт реалізує інтелектуальну систему для автоматизованого аналізу тепловізійних зображень сонячних панелей. Користувач може завантажити зображення фотоелектричного модуля, після чого система виконує обробку зображення, передає його на модель YOLOv11 та повертає результат із виявленими дефектами.

Метою проєкту є підвищення точності та швидкості виявлення дефектів сонячних панелей, що можуть впливати на ефективність їх роботи.

## Основні можливості

* завантаження тепловізійного зображення через вебінтерфейс;
* автоматичне виявлення дефектів фотоелектричних модулів;
* використання навченої моделі YOLOv11;
* відображення результатів детекції на тій самій сторінці;
* показ знайдених дефектів із координатами та рівнем впевненості;
* взаємодія frontend і backend через REST API.

## Принцип роботи

1. Користувач відкриває вебсторінку застосунку.
2. Завантажує тепловізійне зображення сонячної панелі.
3. Frontend надсилає зображення на backend через API.
4. Backend обробляє файл і передає його до моделі YOLOv11.
5. Модель виконує детекцію дефектів.
6. Система повертає результат на frontend.
7. Користувач бачить зображення з результатами виявлення.

## Встановлення та запуск

### 1. Клонування репозиторію

```bash
git clone https://github.com/your-username/your-repository-name.git
cd your-repository-name
```

### 2. Запуск backend

Перейдіть у папку backend:

```bash
cd backend
```

Створіть віртуальне середовище:

```bash
python -m venv venv
```

Активуйте його.

Для Windows:

```bash
venv\Scripts\activate
```

Для Linux або macOS:

```bash
source venv/bin/activate
```

Встановіть залежності:

```bash
pip install -r requirements.txt
```

Запустіть сервер:

```bash
uvicorn main:app --reload
```

Backend буде доступний за адресою:

```text
http://localhost:8000
```

### 3. Запуск frontend

Відкрийте новий термінал і перейдіть у папку frontend:

```bash
cd frontend
```

Встановіть залежності:

```bash
npm install
```

Запустіть frontend:

```bash
npm run dev
```

Frontend буде доступний за адресою:

```text
http://localhost:5173
```

## API

### Перевірка роботи backend

```http
GET /
```

Повертає повідомлення про стан сервера.

### Детекція дефектів

```http
POST /api/predict
```

Приймає зображення та повертає результат роботи моделі.

Приклад відповіді:

```json
{
  "detections": [
    {
      "class_name": "hot cell",
      "confidence": 0.92,
      "box": [120, 85, 250, 190]
    }
  ]
}
```
